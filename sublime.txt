->> What is this?
--> this is sublime build - run watchdogs command with args in vscode.
-> Run Watchdogs inline in Sublime Text
-> Click "Tools" > "Build System" > "New Build System"
put (Windows):
    {
        "cmd": ["windows-native.cmd", "help", "compile"],
        "working_dir": "C:\\Users\\Administrator\\Downloads\\watchdogs"
    }
-> working_dir - location of windows-native.cmd.
 -> example: C:\Users\your_desktop_name\Downloads\watchdogs -> Downloads/Watchdogs
            - put: C:\\Users\\your_desktop_name\\Downloads\\watchdogs
 -> example linux: home/your_desktop_name/Downloads/watchdogs -> Downloads/watchdogs
            - put: /home/your_desktop_name/Downloads/watchdogs
-> cmd - inline commands - [<file>] [<command>] [<args>]
 -> example: "windows-native.cmd", "dir", "/s"
 -> example linux: "watchdogs", "ls", "-a"
 -> example termux: "watchdogs.tmux", "ls", "-a"
-> usage: CTRL & SHIFT + B or CTRL + B
